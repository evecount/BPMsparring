# **App Name**: Digital Spar

## Core Features:

- Camera Integration: Access the user's webcam to track hand movements using mediapipe.
- Combination Display: Display boxing combinations visually on screen. The order of combinations will be decided using a tool (LLM agent) that incorporates the user's recent historical combinations.
- Motion Tracking: Track the user's hands as they strike the air or training equipment using Mediapipe, assessing position and speed.
- Impact Detection: Detect when a hand makes contact with the target areas in the video feed.
- Performance Feedback: Provide real-time feedback on speed, accuracy, and form.
- Leaderboard: Tracks various user stats in the app.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire focus and discipline.
- Background color: Very light blue (#E8EAF6), almost white, for minimal distraction.
- Accent color: Vivid orange (#FF5722) to highlight real-time feedback and actionable elements.
- Body and headline font: 'Inter' for a modern, clean interface focused on readability during intense activity.
- Use clear, simple icons for navigation and stats.
- Clean and focused layout with key feedback elements prominently displayed.
- Subtle animations to highlight correct moves and key moments.
